# sockio
Chat Application built with MEAN stack + Socket.io

### Requirements:
Node, NPM and Mongodb

### Installation:
```sh
$ git clone https://github.com/joel1st/sockio.git
$ npm install
$ npm start
``` 

### Use JShint while developing:
```sh
grunt watch
```

### Before pushing to production minify/concat assets:
```
grunt production
```
